The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Documentation includes each http: request/response routes for the movie_api

URL: http://localhost:8080/movies

GET request to retrieve the entire list of movies. JSON response object format

{ title: "Jaws", description:"", imageURL: "", featured: "yes/no", genre" { name: "horror", description: "", } director: { name: "Steven Speilberg", bio: "a swell guy", }

URL: http://localhost:8080/movies/title

GET req to retrieve each movie by title. JSON response object format,

{ title: "Jaws", description:"", imageURL: "", featured: "yes/no", genre: { name: "horror", dicsription: "gory type of flick", }, director: { name: "Steven Speilberg", bio: "a swell guy", birth year: "", death year: "" }, }

http://localhost:8080/movies/director

GET director by name. returns a JSON object format with info about specific director

{ name: "", bio: "", birth year: "", death year: "", }

http://localhost:8080/movie/genre/:name

GET genre info by name, responds with JSON object format

{ name:"", description:"", }

Query Parameter in URL = :name

http://localhost:8080/users

POST new user to registry expecting a JSON request object.

{ username:"", password:"", email:"", DOB:"", favorite:[], }

http://localhost:8080/users/favorites/id

POST a movie to users "favorites" file. response expecting JSON object from request.

{ username:"", password:"", email:"", DOB:"", favorite:[], }

http://localhost:8080/users/id

DELETE request to remove user from registry by id. response is expecting JSON object from request.

{ username:"", password:"", email:"", DOB:"", favorite:[], }

http://localhost:8080/users/id

DELETE request to remove a specific movie from user's "favorites" by id. response is expecting JSON object from request.

{ username:"", password:"", email:"", DOB:"", favorite:[], }

http://localhost:8080/users/username/email/password/DOB

PUT request to change user info.response is expecting JSON object from request.

{ username:"", password:"", email:"", DOB:"", favorite:[], }

Query Parameters in URL

:username, :email, :password, :DOB
